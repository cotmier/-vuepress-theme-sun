# vuepress-theme-sun

#### Description
VuePress 是一个静态站点生成器
vuepress-theme-sun 是一个简洁、优雅、易于使用的主题，它专注于提供更好的阅读体验。该主题的设计灵感来自于极简主义和现代化设计，采用了深色调的配色方案，非常适合阅读和展示技术文档、博客文章和其他类型的文档。
vuepress-theme-sun 是一个功能齐全、易于使用和美观的 VuePress 主题，非常适合用于创建技术文档和个人博客。
